# bell-experiment-german
Dieser Bericht geht auf die grundlegende Theorie des Bell Experiments ein. Eine Messung vor den Optimierungen wird mit einer Messung danach verglichen: Der Bellparameter bleibt gleich (2.4(0.5)) wobei sich die Zahl der Koinzidenzen verzwanzigfacht.
